Introduction

This section of the documentation describes the extent to which Saxon conforms to external specifications, notably W3C language specifications and Java API specifications.

Statements of conformance are made here in good faith, but since some of the specifications are still changing, the list of discrepancies is almost certainly incomplete. Any information about discrepancies or non-conformances not noted here will be welcomed, and the differences will either be corrected in the product, or documented in a future revision of this section.
